Fill destination image from interpolation of source pixels.
Source pixel coordinate is obtained from invert transformation of destination pixel coordinates.
The default border comportement is ResampleBorderComportement#EXTRAPOLATION.

Moreover : the specified MathTransform should be from CENTER of target image point to CENTER of source image point.
The used MathTransform is consider with PixelInCell#CELL_CENTER configuration.

Resample ( MathTransform  mathTransform,
WritableRenderedImage  imageDest,
RenderedImage  imageSrc,
InterpolationCase  interpolation 
) throws TransformException

Fill destination image area from interpolation of source pixels from imageSrc.
Source pixel coordinates is obtained from invert transformation of destination pixel coordinates.

  • In case where interpolation equals InterpolationCase::LANCZOS lanczos interpolation} the default choosen lanczos window is initialized by value 2.
  • The default border comportement is define as ResampleBorderComportement#FILL_VALUE
  • In case where pixel transformation is out of source image boundary the default choosen fill value is Double#NaN.

Moreover : the specified MathTransform should be from CENTER of target image point to CENTER of source image point.
The used MathTransform is consider as PixelInCell#CELL_CENTER configuration.

Fill destination image area from interpolation of source pixels from imageSrc.
Source pixel coordinates is obtained from invert transformation of destination pixel coordinates.

Moreover : the specified MathTransform should be from CENTER of target image point to CENTER of source image point.
The used MathTransform is consider as PixelInCell#CELL_CENTER configuration. fillValue parameter should be

null

, in case where fillValue is null, when destination coordinates pixel transformation is out of source boundary the destination pixel has no setted sample values.
In other words resampling has no impact on destination pixel samples values when transformation is outside source image boundary.
