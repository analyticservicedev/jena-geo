Detailed Description

An immutable subclass of a Rectangle2D with bounds extending toward infinities.

The getMinX and getMinY methods return always negative infinity, while the getMaxX and getMaxY methods return always positive infinity. This rectangle can be used as argument in the XRectangle2D constructor for initializing a new

XRectangle2D

to infinite bounds.
