Detailed Description

Create metadata objects of the given Class using the properties given in a Map.

For any given type, this class tries to instantiate the metadata object in two ways:

  • First, if a collection of factories has been given to the constructor, then this method will search for a

    create(Map, ...)

    method returning an object assignable to the requested one. For example if the user requests a org.opengis.referencing.crs.VerticalCRS and this

    MetadataFactory

    has been given a org.opengis.referencing.crs.CRSFactory at construction time, then the

    CRSFactory.createVerticalCRS(Map, ...)

    method will be used.

  • If no suitable factory is found, then

    MetadataFactory

    will try to instantiate a metadata implementation directly using Class#newInstance(). The class can be an interface like org.opengis.metadata.citation.Citation or its implementation class like org.apache.sis.metadata.iso.citation.DefaultCitation. The keys in the map shall be the KeyNamePolicy::UML_IDENTIFIER UML identifiers} or the KeyNamePolicy::JAVABEANS_PROPERTY Java Beans name} of metadata properties, for example

    "title"

    for the value to be returned by org.opengis.metadata.citation.Citation#getTitle().
