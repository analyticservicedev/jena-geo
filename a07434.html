Detailed Description

The WKT parser for this authority factory.

This parser add automatically the authority code if it was not explicitly specified in the WKT.

Adds the authority code to the specified properties, if not already present.

In addition, if a primary key was used for fetching the CRS, adds also that primary key to the list of identifiers.
