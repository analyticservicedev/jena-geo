Detailed Description

Key for hints to be specified as a javax.sql.DataSource.

The file may also be specified as a String or Name object. This key also allows null value, which explicitly means "<cite>no data source provided</cite>".

{

Note
Different JNDI implementations build up their name differently, so we may need to look for "jdbc:EPSG" in JBoss and "jdbc/EPSG" in Websphere.}
