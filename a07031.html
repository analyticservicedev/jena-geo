Detailed Description

Replaces Double#NaN values by the weighted average of neighbors values.

This operation uses a box of

size

&times

size

pixels centered on each

NaN

value. The weighted average is then computed, ignoring all

NaN

values. If the number of valid values is greater than

validityThreshold

, then the center

NaN

is replaced by the computed average. Otherwise, the

NaN

value is left unchanged.
