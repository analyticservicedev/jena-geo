ProcessStep getMetadata ( )

Returns a description of the process, the geographic inputs and outputs and other metadata.

Those metadata are suitable to processes on geographic data and may not be applicable to every kind of processes. If this parameter is provided, then:

  • ProcessStep#getDate() is the execution date and time of the process.
  • ProcessStep#getSources() are the geographic Process::getInput() process inputs}.
  • ProcessStep#getOutputs() are the geographic Process::call() process outputs}.
  • ProcessStep#getProcessingInformation() is the Process::getDescriptor() process descriptor}.

◆ rollback()

void rollback ( ) throws ProcessException

Rollback org.geotoolkit.process.Process#call().

Implementers must also handle partial rollback if something when wrong in the middle of org.geotoolkit.process.Process#call().

For example a process that create two files must not fail on rollback because the last file was not created because of the org.geotoolkit.process.Process#call() exception.

Exceptions
ProcessExceptionif something when wrong during rollback.
