Detailed Description

Defines the requirements for a tree node object that can change.

It may changes by adding or removing child nodes, or by changing the contents of a user object stored in the node.

This interface inherits the getUserObject() method from Geotk TreeNode. This is needed because the Swing javax.swing.tree.MutableTreeNode interface defines a setUserObject(Object) method but doesn't define or inherit any

getUserObject()

.
