Detailed Description

Java NIO FileVisitor that copy visited files and directory recursively into target Path.

Usage example : Path sourcePath = Paths.get("/some/path") Path targetPath = Paths.get("/output/path") Files.walkFileTree(sourcePath, new CopyFileVisitor(targetPath));
