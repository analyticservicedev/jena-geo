package

Creates a new objects holding the objects for converting from the given source CRS.

Note about coverage argument: the
coverage
argument is for fetching the coverage CRS when needed. We could get that CRS from GridCoverage#getCoordinateReferenceSystem(), but we use GridCoverage#getCoordinateReferenceSystem() instead for giving users a chance to override. We do not give the coverage CRS in argument because we want to invoke GridCoverage#getCoordinateReferenceSystem() only if
sourceCRS
is non-null, because
getCoordinateReferenceSystem()
may throw IncompleteGridGeometryException.
Parameters
sourceCRSthe source CRS, or
null
if assumed the same than the coverage CRS.
coveragethe coverage for which we are building those information.
gridGeometrythe coverage grid geometry.
Exceptions
TransformExceptionif the crsToGrid transform can not be built.
