Detailed Description

Performs the Flood Fill operation on the given raster.

{

}

This class implements a <cite>Scan line flood fill</cite> algorithm as documented in Wikipedia on June 2009, section <cite>Alternative implementations</cite> (queue-based) modified as described in <cite>Scanline fill</cite> section. The algorithm has been modified in order to work properly with tiled images.

Author
Martin Desruisseaux (Geomatys)
Version
3.01
Since

Process to the Scan Line Flood Fill in one tile.

This method will be invoked for every tiles to process, and may be invoked more than once for the same tile.

Every access on

globalStack

will be performed in a synchronized block. This allow multi-threading, using a different thread for different tiles.
