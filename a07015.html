Default memory size used if System::getProperties() system properties} KEY_CACHE_MEMORY_SIZE} property is not defined.

This value is equivalent to 64MB. Decomposed version : 1024l * 1024 * 1 * 1 * 64 (tile width(px) * tile height(px) * band number * component type size (byte) * number of images.)
