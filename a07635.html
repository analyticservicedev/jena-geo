Detailed Description

The base class for authority factories that create referencing object directly.

This is in contrast with other factories like the AuthorityFactoryAdapter adapter} or CachingAuthorityFactory caching} ones, which delegates their work to an other factory.

Author
Martin Desruisseaux (IRD)
Deprecated:
Moved to org.apache.sis.referencing.factory.GeodeticAuthorityFactory in Apache SIS.
