Detailed Description

The list to be returned by Parameter#values.

This class performs checks on the parameter value to be added or removed. This implementation supports add and remove operations.

boolean add ( final GeneralParameterValue  parameter)

Adds a ParameterValue parameter value} or an other ParameterGroup parameter group} to this group.

If an existing parameter is already included for the same ParameterDescriptor::getName identifier}, then there is a choice:

  • For GeneralParameterDescriptor::getMaximumOccurs maximumOccurs} == 1, the new parameter will replace the existing parameter.
  • For GeneralParameterDescriptor::getMaximumOccurs maximumOccurs} > 1, the new parameter will be added. If adding the new parameter will increase the number past what is allowable by
    maximumOccurs
    , then an IllegalStateException will be thrown.
