Detailed Description

A set of utilities related to Swing.

More utilities are provided in the org.geotoolkit.internal.swing.SwingUtilities static class, which is defined in the

geotk-widgets-swing

module. Here, we have a more minimalist class defined in the

geotk-utility

module only because it is used by other modules that do not depend on the Swing widgets module.

Author
Martin Desruisseaux (IRD, Geomatys)
