Detailed Description

Utilities methods for dealing with exceptions.

Those methods can reformat the stack trace for console output, paint the stack trace in a Graphics2D handler, or show the stack trace in a <cite>Swing</cite> component.

The <cite>Swing</cite> component is available only if the

geotk-widgets-swing

module is available in the classpath. It looks like the following picture:
