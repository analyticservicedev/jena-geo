static<T> T ensureValidValue ( final ParameterDescriptor< T >  descriptor,
final Object  value 
) throws InvalidParameterValueException
package

Ensures that the given value is valid according the specified parameter descriptor.

This convenience method ensures that

value

is assignable to the ParameterDescriptor::getValueClass() expected class}, is between the ParameterDescriptor::getMinimumValue() minimum} and ParameterDescriptor::getMaximumValue() maximum} values and is one of the ParameterDescriptor::getValidValues() set of valid values}. If the value fails any of those tests, then an exception is thrown.

This method is similar to Parameters#isValid(ParameterValue) except that the exception contains an error message formatted with a description of the failure raison.

value

valueThe value to check, or

Returns
The value casted to the descriptor parameterized type, or the ParameterDescriptor::getDefaultValue() default value} if the given value was null while the parameter is mandatory.
