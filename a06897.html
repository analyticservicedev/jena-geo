Detailed Description

An implementation of BufferedImageOp where the result is expected to be an image of the same size than the input.

This class is not public because we rely on JAI when we can. It is used only when an operation can not be easily described in a single JAI parameter block.

{

}

This class must be serializable because the operation may be sent over the network for execution on a remote server.
