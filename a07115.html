Detailed Description

Formats objects in the JSON format.

Author
Martin Desruisseaux (Geomatys)
Todo:
We have an ambiguity when writing
Party.name

: nothing distinguish an individual name from an organization name, since the

Party

type is lost at JSON writing time. We propose to format as

"individual.name"

and

"organisation.name"

in those particular cases.
