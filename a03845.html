Detailed Description

Annotations relative to Java language patterns applied in the Geotk library.

Those annotations are used mostly for documentation purpose. For example a click on the "Use" javadoc link of org.geotoolkit.lang.Configuration gives the list of all static methods having a system-wide effect on the Geotk library.

The javadoc of the org.geotoolkit.lang.Static annotation provides a summary of their "<cite>Use</cite>" page content for convenience.
