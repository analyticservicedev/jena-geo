Detailed Description

The provider for "<cite>New Zealand Map Grid</cite>" (EPSG:9811).

This projection is used with the International 1924 ellipsoid. The math transform implementations instantiated by this provider may be any of the following classes:

  • org.geotoolkit.referencing.operation.projection.NewZealandMapGrid

The following table summarizes the parameters recognized by this provider. For a more detailed parameter list, see the PARAMETERS constant.
