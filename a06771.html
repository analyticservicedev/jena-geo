Detailed Description

The type for keys used to control various aspects of the factory creation.

Factory creation impacts rendering (which is why extending java.awt.RenderingHints.Key rendering key} is not a complete non-sense), but may impact other aspects of an application as well.

{

}

Keys are serializable if the instance to serialize is declared as a public static final constant in the Class::getEnclosingClass() enclosing class}. Otherwise, an java.io.NotSerializableException will be thrown.
