Detailed Description

A path iterator which applies a map projection on the fly.

This iterator extends Point2D.Double for opportunist reasons only - users should not rely on this implementation details.

The point inherited by this class is the location of the last point of the last call to a

currentSegment(...)

method. This location has been transformed by the projection}.

Transforms a line segment from P1 to P2, which may result in a quadratic or cubic curve.

Notation:

  • The (λ,φ) variable names are coordinates in the source space.
  • The (x,y) variable names are coordinates in the target space.
  • The 1 and 2 suffixes denote the starting and ending points respectively.
  • The Δ prefix denote the difference between the target and the source points.
