Detailed Description

Define "Large" RenderedImage which is an image with a large size.


It can contain more data than computer ram memory capacity, in cause of TileCache mechanic which store some image tiles on hard drive.

TODO : Change mecanism to get a source data as entry, not a reader ? It would allow multi-threading on tile reading, by allocating a reader on the fly.

Author
Remi Marechal (Geomatys)
Alexis Manin (Geomatys)
