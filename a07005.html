Detailed Description

Optimized Combine operation for dyadic (two sources) image.

This operation performs a linear combination of two images (

src0

and

src1

). The parameters

scale0

and

scale1

indicate the scale of source images

src0

and

src1

. If we consider pixel at coordinate (x,y), its value is determinate by the pseudo-code:

{ java value = src0[x][y]*scale0 + src1[x][y]*scale1 + offset }
