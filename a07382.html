Default axis info for Easting values in a org.opengis.referencing.crs.ProjectedCRS projected CRS}.

Increasing coordinates values go AxisDirection::EAST East} and units are Units::METRE metres}.

The ISO 19111 name is "<cite>easting</cite>" and the abbreviation is upper case "<var>E</var>".

This axis is usually part of a EASTING, NORTHING set.
