Detailed Description

A writer that put some spaces in front of every lines.

The indentation is initially set to 0 spaces. Users must invoke setIndentation(int) or setMargin(String) in order to set a different value.

Author
Martin Desruisseaux (IRD, Geomatys)
Version
3.18
Since

Deprecated:
Moved to Apache SIS as LineAppender.
