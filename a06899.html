static SampleType valueOf ( final int  bitPerSample,
final int  sampleFormat 
)
static

Mapping between bits per sample and sample format with SampleType.



expected bitPerSamples values :

  • 8(Byte)
  • 16(Short or UShort)
  • 32 (Int or Float)
  • 64 (Double)

expected sampleFormat values :

  • 1 for unsigned integer datas
  • 2 for signed integer datas
  • 3 for IEEE floating point<br>

Example :
UShort data : bitspersample = 16, sampleFormat = 1
Float data : bitspersample = 32, sampleformat = 3

Note : for bitpersample = 8, and bitspersample = 64, sampleFormat is ignored the only available SampleType are respectively SampleType#BYTE and SampleType#DOUBLE. Moreover for each unknow combination bitPerSample, sampleFormat this method return

null

.
