Detailed Description

A trivial implementation of ParameterDescriptor.

This is defined as a subtype of SimpleReferenceIdentifier and SimpleCitation only as an opportunist way (not something generally recommended).

Parameters
<T>The type of value. This type is returned by getValueClass().
