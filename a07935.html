Detailed Description

Implementation of org.w3c.dom.Element to lazy load XML file on element access.

This may be used to reduce memory footprint of javax.imageio.metadata.IIOMetadata when metadata tree is merged with an external metadata xml file.

Author
Quentin Boileau (Geomatys)
