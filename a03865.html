Detailed Description

Read and write objects using the YAML or JSON syntax.

In this library, JSON (<cite>JavaScript Object Notation</cite>) is considered as a subset of YAML.

{

objects}

This package provides static methods accepting an Object argument. The given object shall be an instance of one of the followings:

  • A subclass of org.apache.sis.metadata.AbstractMetadata.
  • An implementation of a GeoAPI interface from the org.opengis.metadata package or sub-packages.
