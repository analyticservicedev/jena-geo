◆ CARTESIAN_2D

final DefaultEngineeringCRS CARTESIAN_2D
static

A two-dimensional Cartesian coordinate reference system with Axes::X x}, Axes::Y y} axes in Units::METRE metres}.

By default, this CRS has no transformation path to any other CRS (i.e. a map using this CS can't be reprojected to a DefaultGeographicCRS geographic coordinate reference system} for example).

◆ GRID_2D

final DefaultImageCRS GRID_2D
static

A two-dimensional Cartesian coordinate reference system with org.geotoolkit.referencing.cs.Axes::COLUMN column}, org.geotoolkit.referencing.cs.Axes::ROW row} axes.

By default, this CRS has no transformation path to any other CRS (i.e. a map using this CS can't be reprojected to a DefaultGeographicCRS geographic coordinate reference system} for example).

The PixelInCell attribute of the associated ImageDatum is set to CELL_CENTER.
