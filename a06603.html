Detailed Description

Applies an arbitrary MathTransform2D (typically a projection) on a Shape.

A

ProjectedShape

instance is a view over a shape, i.e. the shape coordinates are transformed on the fly, never copied. Consequently this class consumes very few memory, but may consume more CPU if the same shape is rendered many time.

{

lines and curves}

Straight line segments may become curves after the transform. This class can create quadratic or cubic curves approximating the original line segments.
