Detailed Description

A DataSource which delegate the execution of every methods to a wrapped} data source.

If non-null username and password have been specified to the constructor, then those authentification info will be used by the getConnection() method.

If addition, this class can also set the read-only state of the connection. This is done in this class because write permissions are sometime determined from the user name (at caller choice).
