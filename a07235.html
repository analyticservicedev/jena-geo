Detailed Description

Base class for thread that will process Reference enqueued in a ReferenceQueue.

This thread invokes Disposeable#dispose on each enqueded Reference reference}. Every Reference implementations to be enqueued in ReferenceQueueConsumer#queue must implement the Disposable interface

Parameters
<T>The type of objects being referenced.
