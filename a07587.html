Detailed Description

The set of codes for a specific type of CRS.

This set filters the codes set in the enclosing WKTParsingAuthorityFactory in order to keep only the codes for the specified type. Filtering is performed on the fly. Consequently, this set is cheap if the user just wants to check for the existence of a particular code.
