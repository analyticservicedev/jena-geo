Detailed Description

Event sent by a running Process process} to its ProcessListener listeners}.

This event contains an optional user-friendly getTask() task description} together with the getProgress() task progression} as a percentage. In addition the event can contains one of the following:

  • A ParameterValueGroup which contain either the final calculation result when the process ProcessListener::completed completed}, or intermediate values while the process is still ProcessListener::progressing progressing}.
  • An Exception which describe either the fatal error why the process ProcessListener::failed failed}, or non-fatal warnings while the process is still ProcessListener::progressing progressing}.

ParameterValueGroup getOutput ( )

Returns the output, which may be an intermediate calculation or the final result.

More specifically:

  • If the event has been given to ProcessListener#progressing(ProcessEvent) then the output (if non-null) is an intermediate calculation.
  • If the event has been given to ProcessListener#completed(ProcessEvent) then the output is the same final result than the one returned by Process#call().
