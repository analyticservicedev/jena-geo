Detailed Description

Prepare a grid traversal tool.

You have to specify at least the polyline to study using one of the setPolyline methods (see setPolyline(double[], int), setPolyline(java.awt.geom.Point2D...) or setPolyline(java.util.List).

You can optionally specify if you want returned point stream to be evaluated with multi-threading using setParallel(boolean) method.

IMPORTANT: There's a method allowwing to configure resulting point stream to omit the polyline first point. That can be useful for ray analysis. By default, result stream return the given polyline start point as first element. Configure this behavior with setIncludeStart(boolean).
