Detailed Description

Holds the host and port number of a URL.

This is equivalent to using the java.net.URI#getHost() and java.net.URI#getPort() methods, Except that no

URI

object is created and no exception it thrown.

This class is useful for URL that are not properly recognized by

URL

or

URI

, like JDBC URL. It is used only for providing default values in widgets.
