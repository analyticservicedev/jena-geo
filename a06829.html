Detailed Description

Convenience static methods for trees operations.

This class provides methods for performing a copy copy} of a tree and for xmlToSwing converting from XML}. It can also #print print} a tree in a format like the following example:

{ text Node #1 ├───Node #2 │   └───Node #4 └───Node #3 }

Author
Martin Desruisseaux (IRD, Geomatys)
Version

Since

Deprecated:
The org.apache.sis.util.collection.TreeTable tree model in Apache SIS} is no longer based on Swing tree interfaces. Swing dependencies will be phased out since Swing itself is likely to be replaced by JavaFX in future JDK versions.

static MutableTreeNode objectToSwing ( final Object  object)
static

Returns a tree representation of the given object.

This method processes the following types especially:

  • If the given object is an instance of Node, then this method delegates to xmlToSwing(Node).
  • If the given object is an instance of Iterable, Map or an array, then a node is created with the name of the implemented interface (for example
    "List"
    or
    "Set"
    ) and each iterated elements is added as a child of the above-cited node.
  • If the given object is an instance of java.util.Map.Entry, then this method returns a NamedTreeNode which contain the entry key as the NamedTreeNode::getName() name} and the entry value as TreeNode::getUserObject() user object}.
  • Otherwise this method returns a single DefaultMutableTreeNode which contain the given object as TreeNode::getUserObject() user object}.

Together with toString(TreeNode), this method provides a convenient way to print the content of a XML document for debugging purpose.
