Detailed Description

A range of grid coverage coordinates, also known as "grid envelope".

GridExtent

are defined by getLow() low} coordinates (often all zeros) and getHigh() high} coordinates, inclusive. For example a grid with a width of 512 cells can have a low coordinate of 0 and high coordinate of 511.

Note: The inclusiveness of getHigh() high} coordinates come from ISO 19123. We follow this specification for all getters methods, but developers should keep in mind that this is the opposite of Java2D usage where java.awt.Rectangle maximal values are exclusive.
GridExtent

instances are immutable and thread-safe. The same instance can be shared by different GridGeometry instances.

Creates a new grid extent by rounding the given envelope to (usually) nearest integers.

The envelope coordinates shall be cell indices with lower values inclusive and upper values exclusive. Double#NaN envelope coordinates will be set to the corresponding

enclosing
