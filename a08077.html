Detailed Description

Find all intersections between a regular grid (whose cells are aligned with 0 origin, and size is 1 on each of their axes) and a polyline.

It returns all points of the polyline, adding intersection points on the fly.

To acquire a stream of all points in the populated line string, use stream(double[], int, boolean) or the Builder class.

This algorithm complexity is roughly O(d*n), with d the number of dimensions of the polyline, and n is the total number of intersection point found.

It is partially splittable : each segment of input polyline will be treated in a separate thread if using a parallel stream. Also, if one of the polyline segment is aligned on an reference axis (meaning only one of the dimension is changing along the segment), it can be splitted too.
