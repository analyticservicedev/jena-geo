Detailed Description

Choose interpolation :

  • neighbor : find nearest pixel values.
  • bilinear : compute biLinear pixel value from nearest 4 pixels values.
  • biCubic : compute biCubic pixel value from nearest 16 pixels values.
  • lanczos : compute biCubic pixel value from nearest n * n pixels values.
    where n = 2 * LanczosInterpolation#lanczosWindow

Note that a margin is defined for each value, specifying a radius in pixel unit. It's the maximal distance used by the interpolation to mix colors. This margin value can be < 0 when the radius is not fixed. Therefore, it's user responsability to define the right value. However, when the margin is negative, it's absolute value should be a clue about a default acceptable margin to apply for this interpolation.
