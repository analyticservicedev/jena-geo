Detailed Description

Methods related to the Geotoolkit.org installation directory.

This is provided for data that need to be saved in a user-specified directory. If the user didn't specified any directory, they will be saved in the temporary directory.

We try to keep the configuration options to a strict minimum, but we still need is some case to specify in which directory are stored the data, for example the NADCON data used for datum shift over United States.

Author
Martin Desruisseaux (Geomatys)

Path directory ( final boolean  usePreferences)

If the preference is defined, returns its value as a Path.

Otherwise returns a sub-directory of the <cite>root directory</cite> where the later is defined as the first of the following directories which is found suitable:

  • ROOT_DIRECTORY user preferences, if defined.
  • ROOT_DIRECTORY system preferences, if defined.
  • ".geotoolkit"
    subdirectory in the user home directory, if the user home directory exists and is writable.
  • "Geotoolkit"
