Detailed Description

FileVisitor implementation with Paths and an input PathMatcher that return all matcher Paths into a list getMatchedPaths().

Input PathMatcher can be test against File only or also with directories using constructor PathFilterVisitor(PathMatcher, boolean). By default PathMatcher is used only with Files type Path.

Quentin Boileau (Geomatys)
