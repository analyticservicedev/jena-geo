URI  resource

static

Update input metadata by adding given URI to its online resources.

If the URI is already present in the metadata, calling this method will have no effect (except consuming CPU time).

Parameters
targetMetadata to update.
resourceThe resource to add as linkage into the metadata.
Returns
True if we successfully added given URI as OnlineResource#getLinkage(). False if we have not because it is already present.
Exceptions
UnsupportedOperationExceptionIf given metadata is read-only.
