Detailed Description

Creates metadata objects of the given Class using the properties given in a Map.

MetadataFactory

tries to instantiate metadata objects using Class#newInstance(). The class given to the create(Class, Map) method is typically a GeoAPI interface (e.g. org.opengis.metadata.citation.Citation), in which case

MetadataFactory

will try to find its implementation class (org.apache.sis.metadata.iso.citation.DefaultCitation). The keys in the map shall be the KeyNamePolicy::UML_IDENTIFIER UML identifiers} of metadata properties, e.g.

"title"

for the value to be returned by org.opengis.metadata.citation.Citation#getTitle().
