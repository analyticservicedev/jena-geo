Detailed Description

ExtendedParameterDescriptor extent the org.apache.sis.parameter.DefaultParameterDescriptor class to add a.

userObject

Map that will contain others additional parameters. Add also a new constructor that take parameter name and remarks and others parameters like validValues, minimum, maximum and units.

Author
Quentin Boileau (Geomatys).
See also
org.apache.sis.parameter.DefaultParameterDescriptor
