Detailed Description

Base class for transformations from a <cite>height above the ellipsoid</cite> to a <cite>height above the geoid</cite>.

This transform expects three-dimensional geographic coordinates in (longitude, latitude, height) order. The transformations are usually backed by some ellipsoid-dependent database.
