Detailed Description

A specialization of GridTransform in the two-dimensional case.

The default implementation is invertible for the OFFSET, NADCON and NTv2 grid types (assuming that the offsets are small), but not for the GridType::LOCALIZATION LOCALIZATION} type. For an invertible localization grid, see the org.geotoolkit.referencing.operation.builder.LocalizationGrid builder.
