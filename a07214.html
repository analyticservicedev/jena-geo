Detailed Description

The coefficients of a polynomial equation.

The equation must be in the form

y = c0 + c1×x + c2×x2 + c3×x3 + ... + cn×xn.

The static method roots(double[]) can be used for computing the root of a polynomial equation without creating a

Polynom

object.
