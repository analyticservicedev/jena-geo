Detailed Description

The method used for creating an instance of some specific class using a factory.

This is used by MetadataFactory only.

Factory method starts with create, and their first parameter type is

Map

,

Map<String,?>

or

Map<String,Object>

.

Author
Martin Desruisseaux (Geomatys)
