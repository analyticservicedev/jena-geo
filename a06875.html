Detailed Description

A filter used during the Trees::copy(TreeNode, TreeNodeFilter) copy} of a tree.

This filter can be used for including only a subset of the original nodes, and to change the org.geotoolkit.gui.swing.tree.TreeNode::getUserObject() user object} assigned to that node.
