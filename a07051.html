Detailed Description

Transforms the sample values for one pixel during a "{@link Combine Combine}" operation.

The method transformSamples is invoked by Combine.computeRect(...) just before the sample values are combined as:

{ java values[0]*row[0] + values[1]*row[1] + values[2]*row[2] + ... + row[sourceBands] }

This interface provides a hook where non-linear transformations can be performed before the linear one. For example, the

transformSamples

method could substitutes some values by their logarithm.
