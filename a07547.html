Detailed Description

The definition of a parameter used by an operation method.

For org.geotoolkit.referencing.crs.AbstractCRS Coordinate Reference Systems} most parameter values are numeric, but other types of parameter values are possible.

For numeric values, the #getValueClass value class} is usually Double, Integer or some other Java wrapper class.

This class contains numerous convenience constructors. But all of them ultimately invoke #DefaultParameterDescriptor(Map,Class,Object[],Object,Comparable,Comparable,Unit,boolean) a single, full-featured constructor}. All other constructors are just shortcuts.
