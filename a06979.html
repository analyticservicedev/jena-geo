Create a Lanczos interpolation.

The Lanczos window is the central lobe of a horizontally-stretched sinc,
sinc(x/a) for |x| ≤ lanczos window.
The normalized sinc function is commonly defined by sinc(x) = sin(PIx)/(PIx).
Lanczos window define interpolate area boundary defined by square of side length 2*lanczos window.

Parameters
pixelIteratorIterator used to interpolation.
lanczosWindowdefine pixel number use to interpolate.
rbcborder comportement.
fillValuevalue(s) which will be filled when coordinate out of source boundary.

Create a Lanczos interpolation.



Define border comportement at ResampleBorderComportement#FILL_VALUE and fillValue is an arrays of the same length than band number from source image and filled by Double#NaN value.

The Lanczos window is the central lobe of a horizontally-stretched sinc,

The normalized sinc function is commonly defined by sinc(x) = sin(PIx)/(PIx).
Lanczos window define interpolate area boundary defined by square of side length 2*lanczos window.
