static float getPixelStride ( final SampleModel  model) throws IllegalArgumentException
static

Returns the <cite>pixel stride</cite> of the given sample model.

The pixel stride is the number of data array elements between two samples for the same band on the same scanline.

If the given model is an instance of MultiPixelPackedSampleModel, then the stride can be fractional. Note that the returned value still exact since the ratio of an integer with a power of 2 has exact representations in IEEE 754.

Parameters
modelThe sample model from which to get the pixel stride.
Returns
The pixel stride, rounded toward positive infinity if necessary.
Exceptions
IllegalArgumentExceptionIf the given model is not of a known type.
