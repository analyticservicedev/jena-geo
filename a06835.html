Detailed Description

A container of image I/O listeners.

This class provides a set of

addFooListener(...)

and

removeFooListener(...)

methods for adding and removing various listeners, and a

addListenersTo(...)

method for copying listeners to the an image reader. This class is convenient when

ImageReader.addFooListener(...)

can't be invoked directly because the ImageReader instance is not yet know or available.

Author
Martin Desruisseaux (IRD, Geomatys)
