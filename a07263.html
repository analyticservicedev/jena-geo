Detailed Description

Read lines of text from various sources like BufferedReader, RandomAccessFile or multi-lines String.

Implementations of this interface accept the Unix and Windows style of <cite>End Of Line</cite> (EOL).

Instances of

LineReader

are created by the static methods defined in LineReaders.
