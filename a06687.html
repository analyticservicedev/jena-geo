Detailed Description

A color model for use with ScaledColorSpace (gray scale image with missing values).

This color model is slightly more efficient than the default ComponentColorModel by reducing the amount of object allocations, made possible by the knowledge that we use only one sample value and returns only one color component (the gray). In addition, this class renders the Float#NaN values as transparent.
