Detailed Description

Manages temporary files created by the Geotk library.

This class provides a createTempFile method which similar to the one provided in the standard File class, except that the file may be deleted earlier. More specifically if the File object has been garbage-collected, then the corresponding file in the file system is deleted immediately instead than waiting for the JVM shutdown.

{

Note
This class extends PhantomReference for implementation convenience. Callers should ignore that detail.}
